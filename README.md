<a name="what"></a>
## Что за белый список интернета?

**Белый список (Whitelist)** — это самый жесткий режим фильтрации (DPI / ТСПУ). В этом режиме оборудование провайдера по умолчанию *запрещает* любой трафик и *разрешает* подключение только к строго определенному списку жизненно важных ресурсов (например, сайты банков, государственные порталы, сервисы Яндекса и VK).

---

<a name="vpn"></a>
## Почему стандартный VPN не помогает

Обычные протоколы, такие как OpenVPN, WireGuard, IKEv2 или IPsec, имеют легко узнаваемые цифровые «отпечатки» (сигнатуры). 
Когда система DPI видит попытку установить такое соединение, она сверяет его со своим списком разрешенных протоколов и доменов. Не найдя совпадений, провайдер просто сбрасывает соединение (Drop). Ваш VPN будет бесконечно писать «Подключение...» и никогда не соединится.
